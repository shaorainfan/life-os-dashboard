关系概况

🧠 原创思维模型：「逻辑是线，情绪是点」

逻辑 = 串联的长线 — 梳理因果、身份、过往经验、行为规律、利弊风险。用来理清"这是谁、发生过什么、安不安全"。

情绪 = 线上的感受节点 — 安心、害怕、抵触、放松、压抑、亲近、疏离。同步共生，一体出现。

长线看逻辑更稳，当下本心看情绪更真。
对外防骗靠逻辑，对内自洽靠情绪。
感性入心，理性立身。

防御模式沟通

  • 📝 女友分享了关于关系被稀释的不安
  • 📝 你感受到的「隔离感、距离感」被讨论
  • 📝 双方的防御模式被识别：她是「佯装模式」，你是「高情绪卷入」

依恋光谱认知

你们的防御模式刚好是同一个依恋创伤光谱的两端。

她的模式：表达脆弱=被拒绝，必须装没事 → 佯装+隔离
你的模式：过度共情+边界松散 → 高情绪卷入

未来方向

  • 🔄 继续直接讨论防御模式，不再猜测和误解
  • 🔄 逐步切换到更真实的沟通模式
